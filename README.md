# Fable Beauté

A premium beauty studio website designed and developed by Timefall Studios.

## Features

- **Modern Design**: A sleek, minimal aesthetic with glass morphism effects.
- **Dynamic Navbar**: Responsive navigation that adapts on scroll.
- **Service Catalog**: Detailed information about our premium treatments.

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)

### Installation

1. Clone the repository and navigate to the project directory.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3009](http://localhost:3009) in your browser.
